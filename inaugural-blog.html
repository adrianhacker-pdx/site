Welcome to my web presence and the inaugural blog entry. For the time being, this site is primarily a professional portfolio. I am pursuing the self-taught software engineer path because I have contributed significantly to the semiconductor industry, and now it's time for something new. Additionally, A.I. is extremely fascinating to me. I see numerous potential applications; A.I. could truly be the gateway to improving the quality of life for many.

A.I. also has the potential to lead humanity in the wrong direction with developments like deepfakes and the ability to spread false messaging convincingly. Could A.I. intersect with blockchain technology to create a public ledger of A.I. media creations? That's an intriguing idea worth exploring further.

In the portfolio section, you can view some of my A.I.-generated art or interact with live-hosted versions of various programming exercises I've completed. In the coming days, I'll be adding a chatbot to this site based on ChatGPT, capable of answering any questions you might have. I'm considering training it with specific knowledge about myself so users can ask various questions about me and receive accurate answers. Time will tell.

The blog section will feature important updates related to technology, tutorials, and other mostly professional topics. I encourage you to use the comment area but please keep it respectful.

These are the key subjects in my self-taught journey, for now:

  • HTML5 / CSS
  • JavaScript
  • Node JS
  • React
  • Docker
  • Amazon Web Services
  • Artificial Intelligence
  • Some Python for web scraping and data visualization

Feel free to reach out if you're moved to do so, interested in hiring me, seeking freelance services, or just generally curious about me.

Cheers,
Adrian
